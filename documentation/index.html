Getting Started

The WeatherCast template is a static HTML template. Not WordPress. If you want to publish/install as HTML, you can edit and publish with any editor (visual studio code, sublime text, notepad++ etc). For this, you need to have a basic understanding of HTML and CSS, but don't worry, the code is well commented and you will be able to modify it easily. When you open HTML files in any editor, you can view and edit texts, add your own images.

No installation is required, to put your site online you just need to connect to your server's FTP and upload your files and your site will work.

Customization

Html

Site consists of 3 sections header, main and footer. Don't worry, the code is well commented and you will be able to modify it easily.

<header>
   [LOGO AND MENU]
</header>

<main>
   <-- help comment -->
   [CONTENT AND SECTIONS OF PAGE]
</main>

<footer>
   [FOOTER CONTENT]
</footer>

Styles

The css/style.css is the main style sheet. This file contains all theme styles like colors, fonts, elements etc. and generic reset (browser interpret the default behavior of html elements differently and with a generic reset CSS we can work around this). Please note that these values may be overwritten elsewhere.

If you want to edit a specific section of the site, simply find the appropriate class in the CSS file, then scroll down until you find the appropriate style you want to edit, then copy the class into the css/custom.css file and make any changes you want to make. This way, with future updates to the theme, you won't lose the changes you've made.

Javascript

The js/main.js file contains all the javascript functions needed to make the theme and its elements work properly.

These files are separated into sections using:

/*--------------------------------------------------------*/
/*	1. NAME OF BLOCK */
/*--------------------------------------------------------*/

Images

All images are located in the imgs folder. If you want to replace an image, just insert it in this folder with the same name as the original file with identical proportions to the image to be replaced.

Scroll animation

To change or add scroll animations just add the data-aos attributes to the element. For more info read the AOS documentation.

Google Maps

To configure the google map enter the street of your company or the name of the Google MyBusiness card by replacing the spaces with the + sign.

Contact Form

To receive and send emails from the contact form of WeatherCast Template you will need to open php/contactform.php file and add your email address.

Notice!

The form will not work unless you have your files running on a hosting server with PHP / JS. The form doesn't work in Localhost but it needs a real server to work.

If you don't receive a message in your mailbox, it could be due to the smtp hosting settings you are using. You can get help with SMTP settings from your hosting company.
